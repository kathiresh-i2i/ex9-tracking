# BrandSpark Marketing Page

Simple static marketing webpage for **BrandSpark** with a **Book a demo** button that collects a user's email and shows a successful submission message (with basic email validation).

## Run

- Open `index.html` in your browser (double-click it), or
- Use any static server (optional).

## What to test

- Click **Book a demo**
- Enter an invalid email → you should see an error message
- Enter a valid email (e.g. `you@company.com`) → you should see a success message and the modal will close
